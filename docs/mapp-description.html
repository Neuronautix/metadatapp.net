A next-generation platform for reliable and efficient preclinical research

MAAP from Ontaya solves one of the biggest bottlenecks in preclinical R&D: scattered, inconsistent, and poorly reusable metadata. MAAP provides a unified, API-first environment where all experimental metadata are captured in a structured, standardized, and interoperable way.

Explore featuresBook a call

Core features

Ontology‑driven metadata

Capture experiments with validated schemas (JSON‑LD + SHACL). Align with Mouse Behavior Ontology (MBO), SEND‑JSON, and lab SOPs for semantic consistency.

API Platform backend

Symfony/API Platform with GraphQL & REST. Auth via Keycloak (OIDC). Postgres storage. Exports to CSV, JSON, JSON‑LD, RO-CRATE, and NWB.

Compliance & ethics

Built-in 3Rs alignment, ARRIVE guidelines, and audit trails. Export for regulatory submissions and ethical reviews.

React admin UI

Fast data entry with smart forms: required fields, controlled vocabularies, date pickers, and data validators.

ML‑ready data

Clean, structured datasets with clear labels and metadata context for downstream analytics, anomaly detection, and model training.

HCM integration

(coming soon) Plug in DVC, LMT, iMouse and other systems. Normalize outputs into comparable behavioral metrics with provenance and versioning.

Standards & Compliance

FAIR Principles

  • Findable: rich metadata/search
  • Accessible: open protocols/auth
  • Interoperable: JSON-LD and standard schemas
  • Reusable: provenance and versioning. Learn more in FAIR-by-Design.

3Rs Alignment

  • Reduction: optimize design
  • Refinement: improve welfare
  • Replacement: support alternatives
  • Ethical oversight and compliance

FAIRRR

Any instance managed through MAAP will be FAIR and thus 3Rs aligned; data choices translate to welfare decisions.
